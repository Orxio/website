# ORXIO Engineering Standards

## Stack
- Next.js 15
- React 19
- TypeScript (Strict)
- Tailwind CSS v4
- shadcn/ui
- Framer Motion

## Principles
- Server Components by default
- Client Components only when required
- Accessibility first
- Mobile first
- SEO optimized
- Reusable components
- Clean architecture
- No inline styles
- Strict TypeScript
- Premium UI
- Enterprise quality

## Naming
- Components: PascalCase
- Hooks: useXxx
- Constants: UPPER_CASE
- Types: PascalCase
- Files: kebab-case (except React components)
